# 资源文件

## 应用图标

请在此目录中放置应用图标文件：

- `icon.png` - 通用图标（至少 512x512）
- `icon.ico` - Windows 图标
- `icon.icns` - macOS 图标

## 图标生成

可以使用在线工具或命令行工具生成不同格式的图标：

- [Icon Generator](https://www.electron.build/icons)
- [electron-icon-builder](https://www.npmjs.com/package/electron-icon-builder)

## 托盘图标

如果需要系统托盘功能，可以添加：

- `tray.png` - 托盘图标（16x16 或 32x32）
- `tray@2x.png` - 高分辨率托盘图标
