## 🔍 查看主进程控制台的方法

### 方法1: 启动应用时开启控制台
在启动应用时加上 `--enable-logging` 参数：
```bash
npm start -- --enable-logging
```
或者：
```bash
npm start
```
然后应用窗口的标题栏会显示控制台选项。

### 方法2: 在开发模式下启动
```bash
npm run dev
```
这样会显示详细的主进程日志。

### 方法3: 通过任务管理器（Windows）
1. 按 `Ctrl+Shift+Esc` 打开任务管理器
2. 找到 "Electron" 进程
3. 右键点击 → "转到详细信息"
4. 右键点击 "electron.exe" → "创建转储文件" 或者直接查看进程输出

### 方法5: 检查应用启动时的控制台输出
启动应用时，终端中应该显示：
```
[IPC] Single-window handlers registered
```

如果看到了这个，说明主进程控制台是活动的。

### ❓ 如果仍然看不到调试信息

请运行以下命令重新启动应用：
```bash
npm start
```

然后立即按 F12，观察终端输出是否有 `[DEBUG]` 开头的日志。
