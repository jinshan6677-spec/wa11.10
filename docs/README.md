### Single-Window Architecture

The new single-window architecture brings:
- **Unified Interface**: All accounts in one window
- **Better Performance**: Optimized memory and CPU usage
- **Faster Switching**: Instant account switching
- **Improved UX**: Modern, streamlined interface

See [Migration Guide](MIGRATION_GUIDE.md) for details.
