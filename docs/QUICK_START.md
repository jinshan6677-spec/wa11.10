# 快速开始指南

## 5 分钟上手 WhatsApp 翻译功能

### 步骤 1: 安装应用

**下载安装包**:
- Windows: 下载 `.exe` 安装程序
- macOS: 下载 `.dmg` 文件
- Linux: 下载 `.AppImage` 文件

**安装**:
- Windows: 双击运行安装程序
- macOS: 拖动到 Applications 文件夹
- Linux: 添加执行权限并运行

### 步骤 2: 启动应用

1. 打开 WhatsApp Desktop Translation
2. 扫描二维码登录 WhatsApp
3. 等待 WhatsApp Web 加载完成

### 步骤 3: 启用翻译

1. 点击右上角的 **🌐 翻译设置** 按钮
2. 启用 **自动翻译** 开关
3. 选择 **目标语言**（例如：中文简体）
4. 点击 **保存设置**

### 步骤 4: 开始使用

**接收消息翻译**:
- 接收到的消息会自动显示翻译结果
- 翻译结果显示在消息下方

**发送消息翻译**:
1. 在输入框输入消息
2. 点击 **🌐 翻译** 按钮
3. 输入框内容会被替换为翻译结果
4. 点击发送

### 步骤 5: 配置翻译引擎（可选）

**使用 Google 翻译**（默认）:
- 无需配置，直接使用

**使用 AI 翻译**（GPT-4、Gemini、DeepSeek）:
1. 获取 API 密钥
2. 在设置中选择翻译引擎
3. 输入 API 密钥
4. 点击 **测试连接**
5. 保存设置

## 常用功能

### 好友独立配置

为不同联系人设置不同的翻译规则：

1. 打开与联系人的聊天
2. 点击 **翻译设置**
3. 启用 **好友独立翻译**
4. 配置目标语言和其他选项
5. 保存

### 禁发中文

防止意外发送中文给外语联系人：

1. 在设置中启用 **禁发中文**
2. 尝试发送中文时会被拦截
3. 需要先翻译才能发送

### 实时翻译预览

输入时即时查看翻译结果：

1. 在设置中启用 **实时翻译**
2. 在输入框输入内容
3. 停止输入后会自动显示翻译预览
